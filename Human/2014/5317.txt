Logic Programming as Scripting Language for Bots in Computer Games -- Research Overview

This publication is to present a summary of research (referred as Klabs - http://www.kappalabs.org) carried out in author's Ph.D studies on topic of application of Logic Programming as scripting language for virtual character behavior control in First Person Shooter (FPS) games. The research goal is to apply reasoning and knowledge representation techniques to create character behavior, which results in increased players' engagement. An extended abstract / full version of a paper accepted to be presented at the Doctoral Consortium of the 30th International Conference on Logic Programming (ICLP 2014), July 19-22, Vienna, Austria
